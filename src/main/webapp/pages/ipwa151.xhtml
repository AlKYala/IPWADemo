IPWA 1-5-1

Hier gehts hauptsaechlich um die Lebensdauer von Beans

Von Links oben nach rechts unten: NoneScoped, Request Scoped, View Scoped, Session Scoped, Application Scoped

None Scoped: Es wird nichts passieren, da nach der Request die Aenderungen sterben

Request Scoped: Sobald du ein neues Request machst, gehen die alten aenderungen weg. (Lebensdauer des Req. endet)

View Scoped: Bleibt so lange wie du den Tab hast.

Session Scoped: Neue Sitzung durch neuen Browser/Incognito Tab

Application Scoped: Bleibt so lange bis der Server down ist
